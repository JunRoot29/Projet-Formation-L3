# Projet Formation - Flask

## Installation

```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
```

## Lancer l'application

```bash
python run.py
```

L'application utilise SQLite par défaut et crée `app.db` automatiquement.

## Utilisateurs de démo

Au premier lancement, deux comptes sont créés :

- Manager: `manager@company.com` / `password`
- Employé: `employee@company.com` / `password`
